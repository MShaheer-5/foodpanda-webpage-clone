# foodpanda-webpage-clone

This project aims to clone the home web page layout of [foodpanda.pk](https://www.foodpanda.pk/restaurants/new?lng=73.03673451876075&lat=33.700414159709915&vertical=restaurants) with the intention of learning and practically utilizing the fundamental concepts of HTML and CSS.

## Features

-Navigation Bar: Allows users to authorize themselves, navigate through different pages and change website language.
